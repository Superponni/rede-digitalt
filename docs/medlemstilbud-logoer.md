# Logo-ønskeliste — Medlemstilbud

For å løfte medlemstilbud-siden fra «pent» til «proft» trenger vi ekte logoer fra
annonsørene. Inntil de er på plass viser kortene bedriftens initialer i
kategorifargen som plassholder.

## Hva vi trenger per logo
- **Format:** PNG eller SVG med **gjennomsiktig bakgrunn** (ikke hvit boks).
- **Oppløsning:** minst ~400 px bred (PNG). SVG er best.
- **Variant:** helst den mørke/farge-versjonen (vises på hvitt kort).

TOBB har som regel disse allerede fra annonsesalget. Filene legges inn per tilbud i
Sanity Studio → **Medlemstilbud** → felt **Logo**.
